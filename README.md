# ⚽ Footy Predictor

A La Liga match prediction app using real-time data and statistical analysis.

## Features
- Live La Liga match data
- AI-powered match predictions with confidence scores
- Team statistics and standings integration
- Predicted scorelines

## Tech Stack
- **Frontend:** React
- **Backend:** Flask, Python
- **Database:** MongoDB
- **API:** Football-Data.org

## Setup
1. Clone the repo
2. Backend: `cd backend && pip install -r requirements.txt && python app.py`
3. Frontend: `cd frontend && npm install && npm start`
4. Add your API key to `backend/.env`
